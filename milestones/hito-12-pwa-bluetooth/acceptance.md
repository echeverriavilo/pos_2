# Hito 12 - Acceptance Criteria

## Validaciones obligatorias

1. Manifiesto PWA válido (manifest.json)
2. Service Worker registrado
3. Caché de assets estático (HTML, CSS, JS, fuentes)
4. Iconos PWA (192x192, 512x512)
5. Theme color configurado
6. Install prompt funcional en Android/iOS
7. Driver de impresión Web Bluetooth
8. Conexión con impresora térmica
9. Impresión de ticket de prueba
10. App funciona offline (UI adaptativo)
