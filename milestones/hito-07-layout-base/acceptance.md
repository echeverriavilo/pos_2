# Hito 07 - Acceptance Criteria

## Validaciones obligatorias

1. Layout base con Bootstrap 5 instalado y configurado
2. Sidebar de navegación lateral responsivo
3. Menú adaptable (móvil: collapse, escritorio: sidebar)
4. Toasts para feedback de sistema (éxito, error, warning)
5. Navegación entre módulos funcional
6. Diseño mobile-first
7. Contenedor principal para contenido
