# Vision — Gastropos

## Objetivo

Gastropos es un sistema de punto de venta (POS) gastronómico, orientado a pequeños negocios como cafeterías, bares y locales de comida rápida.

## Propuesta de Valor

- Bajo costo de adopción
- Mobile-first
- Operación reactiva
- Soporte para múltiples flujos de venta
- Integración flexible con hardware

## Tipo de Sistema

- PWA (Progressive Web App)
- Multitenant (aislamiento por tenant)

## Alcance Funcional

El sistema cubre:

- Gestión de pedidos
- Gestión de mesas (opcional)
- Procesamiento de pagos
- Pagos parciales
- Inventario
- Catálogo de productos
- Roles y permisos por local

## Restricciones

- El sistema debe funcionar en dispositivos móviles
- Debe soportar operación offline (sin definir mecanismo en este documento)
- Debe aislar completamente los datos por tenant
