# Hito 07 - Layout Base y Shell de Aplicación

## Objetivo

Establecer la estructura visual PWA y navegación base.

## Descripción

- Layout base con Bootstrap 5
- Navegación lateral (Sidebar) responsive
- Componentes de mensajes de sistema (Toasts para feedback)
- Navegación funcional entre módulos
- Funciona en móvil y escritorio

## Validación

- Navegación funcional entre módulos vacíos en móvil y escritorio
- Toasts de feedback visibles
- Sidebar responsivo
