# Hito 08 - Terminal de Ventas (Flujo Rápido)

## Objetivo

Interfaz de toma de pedidos de alta velocidad para el flujo rápido.

## Descripción

- Grid de productos con categorías
- Buscador de productos
- Carrito de compra reactivo con HTMX
- Agregar/quitar productos sin recargar página
- Cálculo automático de totales

## Validación

- Agregar/quitar productos del carrito
- Calcular totales sin recargar la página
