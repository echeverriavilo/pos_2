# Hito 11 - Módulo de Pagos y Cierre de Cuenta

## Objetivo

Procesamiento visual de pagos parciales.

## Descripción

- Modal de pago con desglose de IVA/Propina
- Selector de items para pagos parciales
- Ingresos de abonos (monto arbitrario)
- Ticket de pre-cuenta actualizado
- Flujo de cierre de cuenta

## Validación

- Realizar un pago parcial por productos
- Verificar que el ticket de pre-cuenta se actualice correctamente
