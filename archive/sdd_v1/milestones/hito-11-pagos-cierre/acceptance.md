# Hito 11 - Acceptance Criteria

## Validaciones obligatorias

1. Modal de pago con desglose completo (subtotal, IVA, propina, total)
2. Selector visual de items para pago parcial
3. Pago por productos marca items específicos
4. Pago por abono (monto arbitrario)
5. Cálculo correcto de IVA según configuración del tenant
6. Propina opcional con sugerencia (10%)
7. Ticket de pre-cuenta se actualiza tras cada pago
8. Estados de orden actualizan correctamente
9. Transiciones de estado: ABIERTO → PAGADO_PARCIAL → COMPLETADO
