# Hito 12 - PWA, Bluetooth y Offline

## Objetivo

Capacidades nativas y hardware para dispositivos móviles.

## Descripción

- Service Worker para caché de assets
- Manifiesto PWA para instalación en Android/iOS
- Driver de impresión Web Bluetooth para móviles
- Instalación de app en dispositivos
- Impresión de prueba a térmica Bluetooth

## Validación

- Instalar la app en Android/iOS
- Realizar una impresión de prueba desde el celular a una térmica Bluetooth
